```js
 /*
    步骤:
        1.对提交按钮进行处理

        2.给按钮设置点击事件
        3.检测两个输入框 内容是否为空
```

​			模态框 美化: 引用bootstrap.js  和 bootstrap.css ,使用其中的 模态框模块达到美化的 目的.

### 1.2 代码优化

​			将接口调用、用户操作、请求功能,放在同一 *.js 文件下, 引用,  可以方便与后端的对接,  也可以减少后期维护的工作量
